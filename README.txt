# Heartbeat Sensor
Read data input from a generic Photoplethysmography (PPG) heart beat sensor.

This repo can be downloaded as a zip file and imported into the Arduino IDE as a library.

## Installation
1. Download the repo as a .ZIP file
2. In Arduino IDE 2.0, go to 'Sketch > Include Library > Add .ZIP Library...'
3. Navigate to where you downloaded the repo zip file
4. Select the zip file to install
